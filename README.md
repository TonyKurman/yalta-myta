# MYTA — лендинг апартаментов премиум-класса (Ялта)

Одностраничный сайт ЖК **MYTA** с кинематографичным видео-героем: фон —
зафиксированное видео, при скролле блюр плавно уходит, на втором экране текст
печатается по шагам, а сам сайт «врезан» внутрь видео и поднимается снизу
(по образцу обучающего видео).

## Структура

```
myta/
├── index.html              ← рабочий сайт (ассеты подключены ссылками — редактируй этот файл)
├── myta-standalone.html    ← та же страница, всё внутри одним файлом (для быстрого просмотра двойным кликом)
├── assets/
│   ├── myta-hero.mp4        ← фоновое видео героя (16:9). СЮДА кладёшь новый ролик из Seedance
│   ├── poster.jpg           ← постер-кадр (показывается, пока грузится видео)
│   ├── r1.jpg … r8.jpg      ← рендеры (галерея + карточки инфраструктуры)
│   └── plan213.png          ← планировка апартамента (сейчас на сайте не используется)
└── scripts/
    ├── make_clips.sh        ← собрать «живые» клипы из рендеров (ffmpeg, на случай без AI-видео)
    └── build_standalone.py  ← пересобрать myta-standalone.html (вшивает ассеты в один файл)
```

## Как запустить локально

Открыть `index.html` двойным кликом обычно работает. Если видео не
проигрывается из-за ограничений `file://`, подними локальный сервер:

```bash
cd myta
python3 -m http.server 8000
# открой http://localhost:8000
```

`myta-standalone.html` всегда открывается двойным кликом без сервера.

## Как заменить видео в герое (главное)

1. Сгенерируй ролик в **Seedance** (image-to-video) — промпты ниже.
2. Сохрани как `assets/myta-hero.mp4` (16:9, 1080p, h264, без звука).
   При желании обнови `assets/poster.jpg` первым кадром:
   `ffmpeg -i assets/myta-hero.mp4 -frames:v 1 -q:v 4 assets/poster.jpg`
3. Готово — `index.html` уже ссылается на этот путь.

Если есть несколько коротких клипов — склей в одну петлю (см. `scripts/make_clips.sh`
как пример со склейкой `xfade`).

## Дизайн (ориентир для правок)

- Палитра: эспрессо `#1b1410`, сливочный `#f4efe6`, золото `#c2a14e`, камень `#8d7e6b`.
- Шрифты: Cormorant Garamond (заголовки) + Manrope (текст), Google Fonts.
- Эмблема — лотос (инлайн-SVG), повторяется в шапке/контактах.
- Анимации: блюр фона по скроллу, печатание текста (typewriter), reveal-on-scroll,
  врезанная панель контента поверх движущегося видео. Уважается `prefers-reduced-motion`.

## Контент

Тексты сняты с официального сайта `yalta.myta.ru`; данные комплекса — из презентации
проекта. Слайд с апартаментом №213 убран по запросу. Телефон/адрес — в блоке «Контакты».
Форма заявки убрана (вместо неё кликабельный телефон).
